# 🔍 Remote Job Board

A full-stack remote job board that fetches real live job listings from 4 sources updated in real time.

## 🌐 Live Demo
👉 [Click here to view the live app](https://sensational-cascaron-4c7597.netlify.app)

## ✨ Features
- 1000+ real remote jobs from 4 sources
- Search jobs by title, company, skills, location
- Filter by source (Remotive, The Muse, Arbeitnow, Himalayas)
- Filter by job category
- Salary and location badges
- Fully responsive on mobile, tablet and desktop
- Load more pagination

## 🛠️ Built With
- Node.js
- Express.js
- Vanilla JavaScript
- HTML & CSS

## 📡 Job Sources
- [Remotive](https://remotive.com)
- [The Muse](https://themuse.com)
- [Arbeitnow](https://arbeitnow.com)
- [Himalayas](https://himalayas.app)

## 🚀 Run Locally
```bash
git clone https://github.com/mohas1025/Job-board.git
cd Job-board
npm install
node server.js
```
Open `index.html` in your browser.
